char * itoa(long x, char *t)
{
        int i;
        int j;

        i = 0;
        do
        {
                t[i] = (x % 10) + '0';
                x /= 10;
                i++;
        } while (x!=0);

        t[i] = 0;

        for (j=0; j < i / 2; j++) {
                t[j] ^= t[i - j - 1];
                t[i - j - 1] ^= t[j];
                t[j] ^= t[i - j - 1];
        }

        return t;
}

	binary[j] = 0;

				break;
			}

			return 0;
		}

		while (avail > 0) {
			unsigned int bits = (avail < needed) ? avail : needed;
	  
			/* Fill empty space with leftmost bits */
			binary[j] <<= bits;
			binary[j] |= (c>>(6-bits)) & Mask[bits];
			c <<= bits;
			avail -= bits;
			needed -= bits;

				binary[++j] = 0;
				needed = 8;
			}
		}

		binary[i] -= binary[i+1];

MAC Address Spoofing on Kali
The MAC address is a unique combination of numbers and letters with a length of 48 characters. In simple words, it is the hardware physical number of the device (server, PC, router, etc.), which is assigned to the device's network card during production. MAC spoofing is a method of changing/spoofing the MAC address of a network device. MACChanger is a small utility that allows you to change the MAC address of the network card of a computer running the Linux operating system.
MACChanger Parameters:
-h — help output.
-V — display the version.
-s — display the MAC address.
-e — do not change the manufacturer's bytes (the first three bytes).
-a — set a random MAC address of the same type.
-A — set a random MAC address of any type.
-p — reset to original, permanent hardware MAC
-r — set a random MAC address.
-l —list[=STRING] — display information about manufacturers.
-b — pretend to be a stitched address.
To display the network interfaces, open the Kali Linux terminal and enter the command:
ifconfig
We will consider the example of an Ethernet connection, designated as eth0 and connected to a wired LAN. To implement MAC address spoofing, you need to disable the interface:
sudo ifconfig eth0 down
To replace the MAC address (random change), run the command:
sudo macchanger -A eth0
Now you need to enable the eth0 interface:
sudo ifconfig eth0 up
To check the MAC address and the status of the eth0 interface, run the command "ifconfig" again
